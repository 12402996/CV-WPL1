Ik had toen geen ambitie om verder te studeren, omdat ik geld nodig had om te trouwen.

Na 3 jaar werkervaring in de logistiek heb ik besloten om een carrièreswitch te maken. Via VDAB kreeg ik de kans om verder te studeren. Omdat IT mij interesseerde, heb ik gekozen voor de opleiding IT-programmeren. Mijn doel is om later als programmeur aan de slag te gaan.

Ontwikkeling

Waarom heb je gekozen voor deze opleiding?

IT heeft mij altijd geïnteresseerd, daarom heb ik voor deze opleiding heb gekozen. Ik ben in contact gekomen met IT door te gamen. Ik game al sinds ik 10 jaar ben en vroeg me altijd af hoe ik in het spel kon bewegen.

Tijdens mijn opleiding op de middelbare school had ik ervoor gekozen om lessen IT te volgen. Hiervoor moest ik twee uur per week extra lessen volgen. Deze lessen waren vooral gericht op MS-Office. Maar ik wilde de onderliggende code van programma's en games leren kennen.

Na mijn studie kon ik niet verder met IT omdat ik besloten had om te trouwen en hiervoor had ik geld nodig. Tijdens mijn werkervaring ontdekte ik dat je via VDAB een opleiding kon volgen terwijl je een werkloosheidsuitkering ontvangt. Nadat ik werd ontslagen bij mijn laatste werkgever, besloot ik via VDAB een opleiding te volgen genaamd Full-Stack Developer.

Na een gesprek met de VDAB ontdekte ik dat ik ook een IT-gerelateerde opleiding kon volgen aan de PXL, waar de kans op het vinden van werk groter is. Ik kon ook een opleiding systeem- en netwerkbeheer volgen aan de PXL. Maar na het volgen van de onlinecursus Informatica: Programmeren op de site van VDAB, haalde ik hier voldoening uit. Ik heb dan ook gekozen voor de opleiding IT-programmeren aan de PXL.

Waarom denk je dat dit beroep bij je past?

Ik vind het belangrijk om continu kennis op te doen en ik heb de drive om continu bij te leren. Het is geen probleem voor mij om langdurig zittend te werken, zolang ik proactief bezig ben.

Ik merkte dat ik tijdens het werken met de hef- en reachtruck niet meer kon bijleren nadat ik alle taken onder de knie had. Bij mijn werkervaringen had ik vooral soft-skills nodig zoals samenwerken, communiceren en flexibiliteit. Hierdoor ben ik bewust geworden van het belang om ook hard-skills te leren waarbij je continu moet bijleren.

Ik ben een persoon die graag probleemoplossend denkt. Ik zorgde voor een efficiëntere manier van werken op de werkvloer. De mogelijkheden om efficiënter te werken met een hef - en reachtruck zijn beperkt. Hierdoor kreeg ik het gevoel dat ik niets meer kon bijleren. En bij programmeren weet ik dat je continu uitdagingen hebt waardoor je steeds bijleert.

In de IT is het belangrijker om zowel zelfstandig als in teamverband te werken vergeleken met de logistiek als magazijnier. Het lijkt me ook zeer fijn dat ik een probleem op het werk met programmeren ook thuis kan oplossen of op zoek gaan naar oplossingen. Hierdoor blijf ik continu bezig met mijn werk, waardoor ik als programmeur voldoening uit kan halen.
